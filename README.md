# Zikr - Quran Audio Extension

A Chrome Extension for listening to the Holy Quran, built with React, Vite, and Manifest V3.

## Features
- Listen to various Reciters (Sheikhs).
- Browse and play Surahs.
- Background playback support (continues when popup is closed).
- Repeat options (Repeat One / Repeat All).
- English and Arabic language support.
- Dark mode UI.

## Project Structure
- `src/popup`: The React UI for the extension popup.
- `src/offscreen`: The hidden document handling audio playback.
- `src/background`: The Service Worker managing lifecycle and playlist logic.
- `src/services`: API integration with mp3quran.net.

## Setup & Build
1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the extension:
   ```bash
   npm run build
   ```
   This will create a `dist` folder.

## Installation in Chrome
1. Open Chrome and navigate to `chrome://extensions/`.
2. Enable "Developer mode" (top right).
3. Click "Load unpacked".
4. Select the `dist` folder generated by the build.

## Development
To run in dev mode (hot reload):
```bash
npm run dev
```
Note: You may need to reload the extension in `chrome://extensions/` after changes to background scripts or manifest.
